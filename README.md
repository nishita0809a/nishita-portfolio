# Nishita Portfolio Starter Site

This ZIP contains a simple static website structure for Nishita's art and design portfolio.

## Structure

- `index.html` – Home page / overview
- `about.html` – About Me
- `portfolio.html` – Project gallery
- `sketchbook.html` – Sketchbook highlights
- `contact.html` – Contact information
- `assets/css/style.css` – Main styles
- `assets/img/` – Place your artwork images here

## How to Use

1. Open any of the `.html` files in a browser by double-clicking them.
2. Edit the text using a code editor (VS Code, Notepad++, etc.).
3. Replace the placeholder text and hero box with Nishita's real artwork and image files.
4. Upload the contents to any web host or drag into tools like Netlify, GitHub Pages, or your school server.
